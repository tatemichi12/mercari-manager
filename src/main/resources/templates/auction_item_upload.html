入札会商品Excelアップロード

📋 使用方法

  1. テンプレートをダウンロードして、データを入力してください
  2. 必須項目：入札会ID、グループNo.、商品No.（これらは重複判定に使用されます）
  3. 対応形式：.xlsx形式のみ対応しています
  4. 重複処理：既存データは上書き更新されます

📊 カラム構成（24項目）

入札会ID、グループNo.、商品No.、入札額、グループ名、グループ最低入札価格、商品名、備考、商品番号、メーカー、 型式、CPU、CPUモデル、クロック、メモリ、HDD、ドライブ、無線LAN有無、モニタ、COA、 バッテリ状態、タイプ、不良・欠品内容、付属品

📥 Excelテンプレートダウンロード

📤 Excelファイルをアップロード

.xlsx形式のExcelファイルを選択してください

📋 入札会商品一覧へ

⚠️ 注意事項

  • 大容量ファイルのアップロードには時間がかかる場合があります
  • エラー行はスキップされ、処理可能な行のみ登録されます
  • 数値項目（入札額など）は自動的に数値として処理されます
  • 既存データと同じ組み合わせ（入札会ID + グループNo. + 商品No.）のデータは更新されます
